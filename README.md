一个基于AI的数学学习助手，通过引导式提问帮助学生自主解决数学问题。

## ✨ 功能特色

### 🧠 智能引导
- **苏格拉底式提问**：不直接给答案，通过问题引导学生思考
- **个性化教学**：根据学生水平调整问题难度
- **渐进提示**：在学生卡壳时给予适当提示
- **多轮对话**：支持深入的解题讨论

### 📱 多种输入方式
- **图片上传**：拍照或上传数学题目图片
- **文字输入**：直接输入题目内容
- **智能OCR**：自动识别图片中的数学公式和文字
- **知识点分析**：自动识别题目涉及的数学概念

### 💾 会话管理
- **多会话支持**：同时处理多个题目
- **历史记录**：保存所有解题对话
- **进度跟踪**：可视化显示解题进度
- **一键导出**：导出解题过程为文本
